\# NVDA Stock Analysis (2010–2024)



\## 📌 Project Overview

This project analyzes NVIDIA stock price data using Python.



\## 📊 Dataset

S\&P 500 stock dataset (CSV format)



\## 🛠 Tools Used

\- Python

\- Pandas

\- Matplotlib



\## 📈 Workflow

\- Data loading

\- Data cleaning (handling missing values)

\- Return calculation using pct\_change()

\- Data visualization



\## 📊 Key Insights

\- Strong long-term upward trend in NVIDIA stock price

\- High volatility in daily returns

\- Returns show frequent small fluctuations with occasional extreme moves



\## 📁 Project Structure

\- NVDA\_analysis.ipynb → main analysis code

\- figures/ → generated plots

\- data/ → dataset



\## 📌 Conclusion

This project demonstrates basic financial data analysis skills, including data preprocessing, return calculation, and visualization.
